# How to Download Logs

1\. Open Windows PowerShell

<figure><img src="https://cdn-images-1.medium.com/max/800/1*o5uD9amjBsTwLDE844KKHg.png" alt=""><figcaption></figcaption></figure>

2\. Enter the following command in PowerShell: `C:\Users\<YourUserName>\AppData\Local\FluxCore\fluxcore.log`

<figure><img src="https://cdn-images-1.medium.com/max/800/1*BsBErlWwdzh7-m602bLYhA.png" alt=""><figcaption></figcaption></figure>

3\. The log file will be automatically downloaded and opened in your default text editor, such as Notepad.

<figure><img src="https://cdn-images-1.medium.com/max/800/1*UcycBP6hKfPaRUBzcOLMDA.png" alt=""><figcaption></figcaption></figure>
